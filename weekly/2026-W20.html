"Dirty Frag" Linux Kernel LPE Goes Public Before Patches — Root in One Command

Two chained flaws in the kernel's ESP (IPsec) and RxRPC subsystems let any local user escalate to root by retaining write access to page-cache plaintext after decryption. Researcher Hyunwoo Kim published exploit code ahead of schedule after a third party broke the coordinated disclosure embargo, leaving distros scrambling to ship patched kernels. Red Hat, Ubuntu, AlmaLinux, and CloudLinux have all issued updates — if you run Linux servers, patch now before this becomes the next Copy Fail.

Affects kernels since ~4.14 (2017). Public PoC is live. Check your distro's advisory tracker and prioritize patching any internet-facing Linux hosts immediately.

Google Detects First AI-Generated Zero-Day Exploit — Criminals Planned Mass 2FA Bypass

Google's Threat Intelligence Group caught a criminal gang using an LLM to discover and weaponize a zero-day 2FA bypass in a popular open-source sysadmin tool. The Python exploit bore unmistakable AI fingerprints — educational docstrings, a hallucinated CVSS score, and textbook coding structure. The key finding: LLMs can identify semantic logic flaws (hardcoded trust assumptions contradicting security enforcement) that fuzzers and static analyzers fundamentally miss. Google disrupted the operation before mass exploitation, but the implications are stark — AI-assisted vulnerability discovery by criminals has crossed from theoretical to observed in the wild.

Defenders should assume adversaries are using AI for vulnerability discovery. Review your codebases for semantic logic contradictions — the class of flaw traditional scanners miss — and consider how AI-assisted code review could help find them first.

Colorado AI Act Takes Effect June 30 — First State Law Mandating "Reasonable Care" Against Algorithmic Discrimination

Colorado's SB 24-205 kicks in on June 30, making it the first U.S. state to require AI developers to use "reasonable care" to prevent algorithmic discrimination in high-risk systems. Developers must provide deployers a disclosure packet covering foreseeable uses, known risks, and any discovered discrimination incidents — reportable within 90 days to the state AG. The federal government has intervened in a legal challenge to the law, arguing it conflicts with the White House's push for a unified national AI standard, raising the prospect that Colorado's rules could be preempted before they bite.
