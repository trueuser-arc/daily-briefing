Bitwarden CLI npm Package Hijacked — Malware Exfiltrated Tokens, SSH Keys, and Cloud Creds

The @bitwarden/cli@2026.4.0 npm package was compromised on April 22 by TeamPCP as part of the broader Checkmarx supply chain campaign. Malicious code in bw1.js harvested npm tokens, GitHub auth tokens, SSH keys, and AWS/Azure/GCP credentials, encrypting them with AES-256-GCM and exfiltrating via auto-created GitHub repos. Bitwarden confirmed no vault data was accessed and revoked the package within 93 minutes.

Run `npm audit` immediately and rotate any credentials on machines where @bitwarden/cli@2026.4.0 was installed between 5:57–7:30 PM ET on April 22. Pin package versions and enable npm provenance checks.

GitHub App Installation Tokens Switch to Longer Stateless Format Starting Today

Effective April 27, GitHub begins rolling out a new stateless token format for App installation tokens. Tokens grow from the current length to ~520 characters and shift to a ghs_APPID_JWT structure. The staged rollout starts with Actions-issued GITHUB_TOKEN and first-party integrations (Dependabot, Slack, Teams), expanding to all third-party GitHub Apps between mid-May and late June with a brownout period to catch integrations making format assumptions.

If your CI/CD or tooling validates, truncates, or stores GitHub tokens by length, audit those assumptions now. The format change hits Actions workflows first starting today.

U Michigan Maps State Space Models to Compute-in-Memory — Real-Time Edge AI Without Transformers

Researchers at Michigan published in Nature Communications the first implementation of state space models (SSMs) — an emerging alternative to transformers — on a resistive RAM (RRAM) crossbar array. By constraining the SSM to real-number arithmetic, each memory cell directly represents data, enabling real-time video and sensor processing on phones, hearing aids, and autonomous cameras. The 65nm CMOS process means this is fabrication-ready, not just a paper exercise.
