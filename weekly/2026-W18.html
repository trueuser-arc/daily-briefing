Bitwarden CLI npm Package Hijacked — Malware Exfiltrated Tokens, SSH Keys, and Cloud Creds

The @bitwarden/cli@2026.4.0 npm package was compromised on April 22 by TeamPCP as part of the broader Checkmarx supply chain campaign. Malicious code in bw1.js harvested npm tokens, GitHub auth tokens, SSH keys, and AWS/Azure/GCP credentials, encrypting them with AES-256-GCM and exfiltrating via auto-created GitHub repos. Bitwarden confirmed no vault data was accessed and revoked the package within 93 minutes.

Run `npm audit` immediately and rotate any credentials on machines where @bitwarden/cli@2026.4.0 was installed between 5:57–7:30 PM ET on April 22. Pin package versions and enable npm provenance checks.

Microsoft Ships Emergency ASP.NET Core Patch — CVSS 9.1 Flaw Let Attackers Forge Auth Cookies for SYSTEM Access

Microsoft issued an out-of-band security update for CVE-2026-40372, a critical privilege escalation in the ASP.NET Core Data Protection library. A regression in versions 10.0.0 through 10.0.6 caused the managed authenticated encryptor to compute HMAC validation tags over incorrect payload portions — and in some cases discard the hash entirely — allowing unauthenticated attackers to forge authentication cookies and gain SYSTEM privileges. The fix ships in NuGet package version 10.0.7.

If you ship any ASP.NET Core 10 app using Data Protection (which includes most cookie-based auth), update to Microsoft.AspNetCore.DataProtection 10.0.7 immediately, rebuild, redeploy, and rotate your Data Protection key ring to invalidate any forged tokens.
